# Portfolio Website

This is a simple personal portfolio website built using **HTML5** and **CSS3**. It consists of two pages: a **Home** page that introduces me and showcases my portfolio, and a **Contact** page where visitors can find my contact information.

## Features

* Clean and responsive design
* Home page with personal introduction
* Contact page for communication details
* Simple and easy-to-navigate layout
* Built using only HTML and CSS

## Technologies Used

* HTML5
* CSS3

## Purpose

The purpose of this project is to practice the fundamentals of front-end web development by creating a responsive and visually appealing portfolio website using only HTML and CSS.

## Future Enhancements

* Add JavaScript for interactive features
* Include project gallery and resume section
* Add dark mode
* Integrate a functional contact form
* Deploy using GitHub Pages
